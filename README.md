# UiPath Coursework

This repository serves as my centralized learning workspace for **UiPath**. It contains notes, practice exercises, sample workflows, and small automation projects completed while learning through the **UiPath Academy** and other educational resources.

The repository is organized by topic, with each folder dedicated to a specific UiPath concept or feature. Inside each topic, you'll find hands-on exercises, workflow examples, and practice automations that reinforce the concepts being learned.

## Repository Contents

- 📚 Topic-wise learning notes
- 🧪 Practice exercises
- 🔄 Sample UiPath workflows
- 🤖 Small automation projects
- 💡 Concept demonstrations and experiments
- 📝 Learning references and documentation

## Purpose

The primary goal of this repository is to document my learning journey, practice core UiPath concepts, and build a strong foundation in Robotic Process Automation (RPA) using UIPath automation tool. It also serves as a personal reference that I can revisit as I continue to learn and build more advanced automations.

> **Note:** Most of the exercises and workflows in this repository are practice implementations created while following courses from the **UiPath Academy** and other educational resources. They are intended solely for learning purposes and may be adapted or extended to deepen my understanding of the concepts.
